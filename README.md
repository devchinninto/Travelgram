# Travelgram

## Sobre o Projeto

Este projeto foi desenvolvido como parte da **formação Fullstack da Rocketseat**, com o objetivo de praticar e aprimorar habilidades em HTML e CSS. O foco principal foi a criação de um layout moderno e interativo.

Apesar de ter utilizado a base do layout fornecida no curso, editei e personalizei o conteúdo, como as fotos e textos, para transformá-lo em um "travelgram" pessoal, alinhado com a minha paixão por viagens e cultura.

## Acesse a Página

Você pode ver o resultado final do projeto online, sem precisar baixar o código.

-   **Clique aqui para acessar:** [Layout da página de viagens](https://devchinninto.github.io/Travelgram/)

## Funcionalidades e Layout

-   **Bio do Viajante:** Uma seção dedicada à apresentação pessoal.
-   **Galeria de Fotos:** Exibição de imagens de viagens em um formato de grade.
-   **Estrutura semântica:** Uso de HTML5 para uma estrutura clara e acessível.

## Tecnologias Utilizadas

-   `HTML5` para a estrutura da página.
-   `CSS3` para a estilização, incluindo técnicas de flexbox e grid para o layout responsivo.
